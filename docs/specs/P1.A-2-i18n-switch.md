## 1. 背景与动机

用户需要在中文和英文之间切换界面语言。参照设计稿，点击 Globe 图标弹出下拉菜单，选择 "简体中文" 或 "English"。

## 2. 范围

### 2.1 In Scope

- `LocaleToggle` 下拉菜单组件
- `apps/web/src/lib/i18n.ts`：翻译字典 + useTranslation hook
- 首页文案国际化
- 接入 HomeHeader 的 Globe 按钮

### 2.2 Out of Scope

- next-intl 完整集成（Phase 2，需要 server-side i18n）
- 非首页页面的 i18n

## 3. Artifacts

| File                                       | Description          |
| ------------------------------------------ | -------------------- |
| `apps/web/src/lib/i18n.ts`                 | 翻译字典 + useT hook |
| `apps/web/src/components/LocaleToggle.tsx` | 语言切换下拉         |

## 4. 验收标准

- **AC-1**: 点击 Globe 弹出 "简体中文" / "English" 两项
- **AC-2**: 选择后首页文案切换
- **AC-3**: `pnpm check` 全绿
