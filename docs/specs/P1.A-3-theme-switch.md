## 1. 背景与动机

用户需要切换 light / dark / system 主题。参照设计稿 `01-首页-主题切换.png`，点击工具栏月亮图标弹出下拉菜单。

## 2. 范围

### 2.1 In Scope

- `ThemeProvider`：监听 settingsStore.theme，同步到 `<html>` class
- `ThemeToggle`：下拉菜单组件（浅色 / 深色 / 跟随系统）
- 接入 HomeHeader 的主题按钮

### 2.2 Out of Scope

- localStorage 持久化（Phase 1 后续，当前用 Zustand 内存态）

## 3. Artifacts

| File                                        | Description              |
| ------------------------------------------- | ------------------------ |
| `apps/web/src/components/ThemeProvider.tsx` | script 注入 + class 同步 |
| `apps/web/src/components/ThemeToggle.tsx`   | 下拉菜单                 |
| `apps/web/src/app/layout.tsx`               | 包裹 ThemeProvider       |

## 4. 验收标准

- **AC-1**: 点击主题按钮弹出 3 个选项
- **AC-2**: 选择后 `<html>` class 切换为 `dark` 或移除
- **AC-3**: "跟随系统" 根据 `prefers-color-scheme` 自动切换
- **AC-4**: `pnpm check` 全绿
