## 1. 背景与动机

课堂播放引擎和 Agent trace 面板需要跨组件共享状态。Phase 0 只建骨架（完整 shape + 空 action），Phase 1 填入真正的 reducer 逻辑。

## 2. 范围

### 2.1 In Scope

- `apps/web/src/stores/lesson-store.ts`：当前课程状态
- `apps/web/src/stores/playback-store.ts`：播放状态机
- `apps/web/src/stores/trace-store.ts`：Agent trace 收集

### 2.2 Out of Scope

- Playback 状态机 reducer（P1.C-2）
- Trace panel UI（P1.D-1）

## 3. Store Shapes

### lesson-store

```typescript
interface LessonStoreState {
  lesson: Lesson | null;
}
interface LessonStoreActions {
  setLesson: (lesson: Lesson | null) => void;
  reset: () => void;
}
```

### playback-store

```typescript
interface PlaybackStoreState extends PlaybackState {}
interface PlaybackStoreActions {
  setStatus: (status: PlaybackState["status"]) => void;
  reset: () => void;
}
```

### trace-store

```typescript
interface TraceStoreState {
  traces: TraceRecord[];
}
interface TraceStoreActions {
  addTrace: (trace: TraceRecord) => void;
  clearTraces: () => void;
}
```

## 4. 验收标准

- **AC-1**: 3 个 store 有完整 TypeScript shape + Immer middleware
- **AC-2**: 每个 store 至少 2 项测试
- **AC-3**: `pnpm check` 全绿
