# Spec: 本地 CI 脚本

| 字段          | 值             |
| ------------- | -------------- |
| **Spec ID**   | `P0.C-6`       |
| **状态**      | approved       |
| **所属阶段**  | Phase 0        |
| **依赖 spec** | P0.A-5, P0.C-4 |

---

## 1. 背景与动机

在 GitHub Actions CI 接入之前（Phase 2），需要一个本地等价的 CI 脚本，开发者推代码前手动运行确保质量。

## 2. 范围

### 2.1 In Scope

- `scripts/ci-local.sh`：依次运行 format:check → lint → typecheck → test → build

### 2.2 Out of Scope

- GitHub Actions 配置（Phase 2）
- E2E 测试（Phase 2）

## 3. 脚本内容

```bash
#!/usr/bin/env bash
set -euo pipefail

echo "=== [1/5] Format check ==="
pnpm format:check

echo "=== [2/5] Lint ==="
pnpm -r lint

echo "=== [3/5] Typecheck ==="
pnpm -r typecheck

echo "=== [4/5] Test ==="
pnpm -r test

echo "=== [5/5] Build ==="
pnpm --filter web build

echo "=== ✅ All checks passed ==="
```

## 4. 验收标准

- **AC-1**: `./scripts/ci-local.sh` 全绿
- **AC-2**: `pnpm check` 全绿
