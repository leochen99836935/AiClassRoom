# AIClassRoom

企业级 AI 互动课堂平台。**文档驱动 / Vibe Coding** —— 所有代码由 Claude Code 基于 `docs/` 生成与维护。

> 文档是唯一真相来源，代码是文档的投影。

## 快速开始

```bash
pnpm install
pnpm dev
```

| [docs/PLAN.md](./docs/PLAN.md)                 | 分阶段开发计划                  |
| [docs/specs/](./docs/specs/)                   | 每个功能的详细规范              |
